Context and Purpose

In completing this assignment, I used AI tools to assist me with tasks such as conducting interviews, troubleshooting errors, and exploring content ideas, typography, and layout. I also used them to help paraphrase sentences used on this website.

Critical Decision-Making and Ownership

When exploring typography and layout, I selected the layout or typography I was looking for and ignored the rest of the code. I also immediately typed a prompt, "How to highlight text in HTML?" Copilot provided several suggestions, including inline CSS and CSS classes. I used the CSS classes.

To explore the content, I looked at it structurally and adapted it to my website. I discarded some details and only took the main points. The content, especially for the Accessibility Assessment and Ethics Assessment, still refers to the original website guidance for WCAG 2.2 and the Week 2 Lecture Slide (Ethics Assessment). With Copilot's help, I paraphrased the text to improve my grammar.

Accuracy, Bias, and Ethical Awareness

AI provides details of the content regarding the WCAG 2.2 Guidelines. I double-checked the information provided on the website "www.w3.org/TR/WCAG22" for compliance. The information is consistent, but some information needs to be rewritten to improve readability.

Privacy and Sensitive Contents

No sensitive or private information was disclosed in any prompt. The user information in the User Storie Page are imaginary and come from imaginary character.

Reflective on Learning

Using AI tools helps me identify issues in my code, providing code recommendations based on the layout I outlined in the prompt. However, knowing that sometimes the code provided isn't accurate, I realize the importance of continuing to experiment and evaluate before deciding to use the AI's code suggestions.

Attribution and Final Statement

AI tools helped with some stages of my approach, such as exploring, ideating, and paraphrasing. I made all of the decisions on content, language, and alignment with user needs. The final submission is a reflection of my own study, principles, and comprehension of ethical, accessible design.
